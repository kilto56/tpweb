La programmation en JavaScript

Le JavaScript (Js) et son histoire...

Le Javascript est un language crée en 1995 par Brendan Eich qui sert à animer des pages et sites web. Il sert également à effectuer des événements, actions, calculs, détections d'actions du client etc...
Ce language a subi plusieurs mises à jour depuis sa création. La plus récente est l'ECMA script 6 qui a permis au développeur d'utiliser une syntaxe plus rapide. Elle a aussi permis de régler différents problèmes comme celui de la déclaration de variables avec "var".

number = 2;
var number = 3;



La syntaxe du JavaScript

La syntaxe de ce language de programmation est en réalité assez simple mais un caractère manquant peut faire basculer tout votre résultat ! Il faut donc y preter attention !
En effet en Js, il suffit de retenir comment organiser son code pour connaitre toute sa syntaxe ! Exemple : "=" pour attribuer une valeur, "," pour ajouter un paramètre, "{" pour ouvrir un bloc de code...

let variable = "string";
for (i = 0; i <= 3; i++) {
console.log(variable);
};
