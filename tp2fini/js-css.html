La programmation en JavaScript

Liaison CSS - Js

Le JavaScript possède une syntaxe qui permet d'utiliser du style dans une page web. Il suffit de séléctionner l'élément grace à "querySelector" ou "getElementById/TagName..." ou encore "querySelectorAll" (et j'en passe)...

document.querySelector("p").color = "red";

Cliquez sur les différents bouttons ci dessous pour modifier la couleur du rectangle.
