Liaison HTML - Js

Le JavaScript peut se lier à une page HTML très simplement ! Soit interne soit par recherche d'une feuille différente !
On utilise donc :
‹script›
//le code en JavaScript
‹/script›.
Ou :
‹script src="script.js"›
‹/script›
Il vaut mieux utiliser la seconde méthode pour privilégier une meilleure lecture de votre site web par le navigateur et toujours mettre ces balises en fin de body car il faut lancer le JavaScript en dernier dans une page pour éviter les dégats.

‹!DOCTYPE html›
  ‹html›
    ‹body›
      ‹h3›Lier une feuille Js ‹/h3›

      ‹script src="script.js"›
      ‹/script›
    ‹/body›
  ‹/html›

Exemple

Inserez une valeur à l'input et cliquer sur les boutons pour faire réagir la page.
